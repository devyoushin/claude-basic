`$ARGUMENTS`를 파싱합니다:
- 첫 번째 인자: 대상 파일 경로
- 나머지: 추가할 문제 또는 팁

## 작성 형식

```markdown
### {문제 또는 팁 제목}

**상황**: {어떤 상황에서 발생하는지}

**잘못된 방법** ❌:
```
{나쁜 프롬프트 예시}
```

**올바른 방법** ✅:
```
{좋은 프롬프트 예시}
```

**토큰 차이**: ~{X}토큰 절약/추가
**관련 규칙**: `rules/{파일}.md` — {규칙 ID}
```

추가 완료 후 `tokenops/log.md`를 업데이트해 주세요.
